# 1. 项目说明
1. 仿PC端小米商城，采用vue.js(2.x)+elementUI开发。
2. 主要包含页面：商城首页、商品详情页、购物车页、下单页、登录页。
3. 代码分支：master分支为：静态页面+接口交互
4. 前端框架：vue2.x。
5. UI框架：elementUI。
